# 하남 교산지구 · 교통약자 맞춤형 교통안전 분석

> 점·선·면 공간 데이터를 100m 격자로 통합해 교통 위험도를 진단하고, 토지이용계획 기반으로 미래 위험을 예측하여 스마트 안전 시설물의 최적 입지를 제안한 공간 데이터 분석 프로젝트.

`Python` · `GeoPandas` · `scikit-learn` · `statsmodels` · `SciPy` · `QGIS` · `WGS84(EPSG:4326)` · `EPSG:5179`

---

## 1. 프로젝트 개요

3기 신도시(하남 교산)는 아직 사고 데이터가 쌓이지 않은 지역입니다. 이 프로젝트는 **이미 조성된 인접 신도시**(동탄·위례·하남미사·판교)의 데이터로 위험 구조를 학습한 뒤, 이를 하남 교산의 **토지이용계획**에 투영해 "아직 일어나지 않은 위험"을 선제적으로 진단하는 것을 목표로 합니다.

분석은 네 단계의 노트북으로 구성됩니다.

| 단계 | 노트북 | 역할 |
|------|--------|------|
| 01 | `01_Data_Preprocessing.ipynb` | 이종 공간 데이터를 100m 격자로 통합 |
| 02 | `02_Feature_Engineering.ipynb` | 4대 위험 지수 설계 → 현재 위험도 산출 |
| 03 | `03_Main_Analysis.ipynb` | 미래 위험도(FRI) 예측 · 어린이 사고 군집분석 |
| 04 | `04_Conclusion.ipynb` | 페르소나 매칭 → 스마트 시설물 최적 입지 6선 도출 |

## 2. 분석 파이프라인

```mermaid
flowchart LR
```

### STEP 01 · 데이터 전처리 & 격자 통합 (`01_Data_Preprocessing.ipynb`)

- **데이터 수집**: COMPAS 공모전 API(`geoband.API.GetCompasData`)로 격자·인구(거주/유동/직장/방문/서비스)·도로망·속도·교통량·혼잡강도·사고이력·시설(학교/유치원/어린이집/보호구역)·시설물(횡단보도/버스/CCTV/과속방지턱)·토지이용계획도 등 25종을 적재.
- **격자 집계 파이프라인** (`make_gid_agg`): 위경도 포인트를 `EPSG:5179`로 변환 후 격자에 `within` 공간 조인하여 `gid` 단위로 집계. 인구·시설 데이터를 일괄 처리.
- **도로(선) → 격자(면) 길이가중 결합**: 도로망과 격자를 `gpd.overlay(how='intersection')`로 교차해 세그먼트 길이(`seg_length`)를 산출하고, 속도저하율·혼잡강도·교통량을 길이가중·면적보정해 격자별 혼잡 지표로 환산.
- **시설물 격자 집계** (`count_in_grid`): CCTV·횡단보도·방지턱·버스정류장을 격자별 개수로, 어린이보호구역은 `intersects` 플래그로 매핑.
- **외부 데이터 보강**: 국토통계(국토정보맵) 100m 격자 인구(고령·유소년·유아, 4개 지역, 2024년 4월·10월)를 SHP에서 중심점 추출 후 4·10월 평균으로 통합 → 어린이 거주인구 지표(`grid_kukto`) 생성.
- **신뢰성 검증**: 외부 추정 인구와 공모전 제공 거주인구(고령)를 좌표로 매칭해 상관계수를 확인(보고값 ≈ 0.99) — 외부 데이터 활용의 타당성 확보.

### STEP 02 · 4대 위험 지수 & 현재 위험도 (`02_Feature_Engineering.ipynb`)

**① 사고지수** — `사고심각도(EPDO) × 약자가중치 × 법규위반가중치` 후 `log(1 + Σ점수)` → MinMax 정규화.

- EPDO: 사망 12 · 중상 5 · 경상 3 · 부상신고 1 (물피 최소 1점)
- 약자가중치: 12세 이하 1.3 · 65세 이상/61–64세 1.2 · 그 외 1.0
- 법규위반: 신호·보행자보호·중앙선 1.5 · 교차로 운행방법 1.3 · 안전거리 미확보 1.2 · 그 외 1.0

**② 교통혼잡지수** — `(정규화 속도저하율 × 교통부하) + 정규화 혼잡강도`. 혼잡강도는 `log1p` 후 MinMax. 격자별 시간대 평균(`mean_congestion_index`)으로 집계.
→ *상위 5% 격자가 전체 혼잡 가중치의 약 23.5%를 점유*하는 집중 구조를 통계로 확인.

**③ 취약수요지수(VDI)** — 어린이 유동인구 통계 공백을 **시설 인가 정원**으로 추정(초등 `AVG_ELEM_CAP = 600`, 유치원·어린이집은 인가정원/현원). 취약비율(`v_ratio`)을 연령정보 없는 인구(방문·직장·서비스)에 비례 할당.
`VDI = Σ취약활동인구 / 전체활동인구` → `log1p` → 99% 클리핑 → MinMax. (서비스 인구는 중복 보정계수 0.5 적용)

**④ 안전시설미비지수** — `(CCTV+횡단보도+방지턱) × (1 + 0.4 × 보호구역플래그)`를 MinMax 정규화한 시설지수의 역수(`1 − 시설지수`).

**현재 위험도 통합** — 취약계층(12세 이하·61–64·65세 이상) 사고 발생 여부(0/1)를 타깃으로 **개별 로지스틱 회귀**를 돌려 변수 유의성을 검정하고, **회귀계수 절대값**을 환경 가중치(합 0.6)의 근거로 채택(엔트로피법 미사용). 사고지수는 0.4로 고정.

```
현재위험도 = w1·혼잡 + w2·시설미비 + w3·취약수요 + 0.4·사고지수   (w1+w2+w3 = 0.6)
```

→ 위험 판별 기여도는 **취약수요 ≈ 52% vs 시설미비 ≈ 0.6%**. "시설물 개수"보다 "취약계층 밀집도"가 위험을 가른다는 결론. (시설미비지수는 판별 기여도는 낮지만 *시설 설치 우선순위* 결정용 정책 변수로 모델에 유지)

### STEP 03 · 미래 위험도(FRI) & 군집분석 (`03_Main_Analysis.ipynb`)

- **보정계수 F 설계**: 하남 교산 토지이용계획의 용도(`blockType`)·명칭(`blockName`)별 노출 점수 부여 — 교육시설(유·초·특) +2.0, 주상복합 +1.5, 공동주택 +1.0, 중·고교 +0.5, 녹지·공원은 음수(안전 보정). 공공공지는 도로(<5m)·안전시설(<5m) 인접성으로 가감. 학교 100m/300m 버퍼로 보행 노출 가중.
- **FRI 산출**: `FRI = 현재위험도 × (1 + MinMax(F))`
- **신뢰성 검증**: 현재 위험 순위 vs 미래 위험 순위 **Spearman 상관 ≈ 0.7251**(`scipy.stats.spearmanr`) — 기존 위험을 계승하면서 신규 잠재 위험 구역을 발굴함을 정량 검증. 순위 급상승 격자를 잠재 위험 구역으로 식별.
- **어린이 사고 군집분석**: '차대사람 & 12세 이하' 사고를 추출해 정성 변수(요일·시간·위반·도로형태·날씨·가해차종 등)는 원-핫 인코딩, 수치 변수(시설미비·현재위험도)는 표준화. **엘보우 기법(K=2~12) → K=4** 선정 후 `KMeans` 군집화, EPDO 심각도와 결합.

### STEP 04 · 페르소나 매칭 & 시설물 최적지 (`04_Conclusion.ipynb`)

- **하이브리드 페르소나**: 도로망에서 링크 3개 이상이 만나는 **교차로 노드**를 추출하고, 격자의 교차로 여부 + 지수 유사도(현재위험도·시설미비 중심값 거리)로 두 페르소나로 매칭 — **C0 단일로/시설미비형**, **C1 교차로형**.
- **시설 매칭**: C0 → *스마트 통합 패키지*(AI 감응형 횡단보도 + 감응형 방지턱), C1 → *AI 감응형 스마트 횡단보도*.
- **정량 우선순위 산출**: 군집 페르소나 평균(심각도·혼잡도)에 격자 상대 가중치(`W_i = FRI / 군집평균FRI`)를 곱해 잠재 사고량·혼잡도를 환산. 시간가치 계수 `K=15`(EPDO 1점 = 15분 정체 예방), 시설별 사고감소율(C0 0.8425 / C1 0.25)을 적용해 **안전 편익(ΔE)·소통 편익(ΔFlow)**을 산출·정규화·합산.
- **결과**: 군집별 상위 3개씩 **최종 6개 우선 설치 후보지**와 정량 선정 사유 자동 생성.

## 8. 한계 및 향후 과제

- 어린이 유동인구는 직접 관측치가 아닌 **시설 인가 정원 기반 추정값**이므로, 실측 유동 데이터 확보 시 정밀도 향상 여지가 있음.
